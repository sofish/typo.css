⚔️ 项羽本纪

司马迁 · 《史记》卷七 · 约公元前 206 年

项籍者，下相人也，字羽。初起时，年二十四。其季父项梁，梁父即楚将项燕，为秦将王翦所戮者也。项氏世世为楚将，封于项，故姓项氏。

项籍少时，学书不成，去；学剑，又不成。项梁怒之。籍曰："书足以记名姓而已。剑一人敌，不足学，学万人敌。"

深度解析：从"万人敌"到"四面楚歌"

项羽是《史记》中唯一未称帝而入本纪的人物，司马迁此举打破了"本纪记帝王"的体例约束。项籍少年时的"万人敌"宣言，展现了他不甘平凡的雄心，而这种项羽式自信——或者说傲慢——贯穿了他的一生。

巨鹿之战：破釜沉舟的军事天才

项羽乃悉引兵渡河，皆沉船，破釜甑，烧庐舍，持三日粮，以示士卒必死，无一还心。于是至则围王离，与秦军遇，九战，绝其甬道，大破之，杀苏角，虏王离。

巨鹿之战是项羽军事生涯的巅峰。他创造性地运用心理战术——"破釜沉舟"断绝退路，将士兵的恐惧转化为必死的决心。历史上对此战的实际兵力对比一直有争议近年考古发现证实，项羽军约五万对秦军二十万，是典型的以少胜多。

鸿门宴：政治智慧的反面教材

鸿门宴暴露了项羽性格中致命的弱点。范增"数目项王，举所佩玉玦以示之者三"，但"项王默然不应"。项羽不是没有机会杀刘邦——他有三次——但贵族出身的傲慢让他不屑于用暗杀手段解决政治对手。

四面楚歌与垓下绝唱

项王军壁垓下，兵少食尽，汉军及诸侯兵围之数重。夜闻汉军四面皆楚歌，项王乃大惊曰："汉皆已得楚乎？是何楚人之多也！"

于是项王乃悲歌慷慨，自为诗曰："力拔山兮气盖世，时不利兮骓不逝。骓不逝兮可奈何，虞兮虞兮奈若何！"

《垓下歌》仅四句二十八字，却浓缩了英雄末路的全部悲怆。力能拔山的西楚霸王，最终败给了自己的性格——刚愎自用、不善用人、缺乏政治远见。司马迁用"岂不谬哉"四字作结，不直接批评，却比任何批评都有力量。

👑 高祖本纪

司马迁 · 《史记》卷八 · 约公元前 195 年

高祖，沛丰邑中阳里人，姓刘氏，字季。父曰太公，母曰刘媪。其先刘媪尝息大泽之陂，梦与神遇。是时雷电晦冥，太公往视，则见蛟龙于其上。已而有身，遂产高祖。

深度解析：神化叙事与帝王合法性

司马迁对刘邦的记载充满了荒诞的神话色彩——"蛟龙于其上"而孕、"常有大度"、醉卧时"其上常有龙"、所居"常有云气"。这些神迹叙事并非司马迁迷信，而是一种极为精妙的政治书写策略。

从"无赖"到皇帝的逆袭

高祖常繇咸阳，纵观，观秦皇帝，喟然太息曰："嗟乎，大丈夫当如此也！"

与项羽的"彼可取而代也"不同，刘邦说的是"大丈夫当如此"——不是取而代之，而是羡慕与向往。这一句话暴露了两个人的本质差异：项羽要取代，刘邦要成为。

刘邦出身平民却建立了四百年基业，其成功绝非运气。他最大的天赋是能用比自己强的人——萧何、张良、韩信、陈平、曹参……几乎每一个关键人物都比刘邦在某方面更强。刘邦自己总结：

夫运筹策帷帐之中，决胜于千里之外，吾不如子房。镇国家，抚百姓，给馈饷，不绝粮道，吾不如萧何。连百万之军，战必胜，攻必取，吾不如韩信。此三者，皆人杰也，吾能用之，此吾所以取天下也。

大风歌：功成之后的落寞

高祖还归，过沛，留。置酒沛宫，悉召故人父老子弟纵酒，发沛中儿得百二十人，教之歌。酒酣，高祖击筑，自为歌诗曰："大风起兮云飞扬，威加海内兮归故乡，安得猛士兮守四方！"

《大风歌》与项羽的《垓下歌》形成了史记中最震撼的双雄对照。一个在成功时唱出了不安（"安得猛士兮守四方"），一个在失败时唱出了不甘（"时不利兮骓不逝"）。司马迁用这两首歌，为楚汉争霸画上了一个文学意义上的句号。

🔥 陈涉世家

司马迁 · 《史记》卷四十八 · 约公元前 209 年

陈胜者，阳城人也，字涉。吴广者，阳夏人也，字叔。陈涉少时，尝与人佣耕，辍耕之垄上，怅恨久之，曰："苟富贵，无相忘。"佣者笑而应曰："若为佣耕，何富贵也？"陈涉太息曰："嗟乎，燕雀安知鸿鹄之志哉！"

深度解析：为何陈涉能入"世家"？

陈涉是中国历史上第一个发动大规模农民起义的人。司马迁将其列入"世家"——通常记载诸侯王——而非"列传"，足见对其历史地位的高度评价。

二世元年七月，发闾左适戍渔阳，九百人屯大泽乡。陈胜、吴广皆次当行，为屯长。会天大雨，道不通，度已失期。失期，法皆斩。陈胜、吴广乃谋曰："今亡亦死，举大计亦死，等死，死国可乎？"

"今亡亦死，举大计亦死，等死，死国可乎？"这句话浓缩了秦末底层人民走投无路的绝望。司马迁用了不到一百字，就勾勒出了从"会天大雨"到"揭竿而起"的全部因果链。这种叙事效率，两千年来无人能及。

"天下苦秦久矣"——一则完美的起义檄文

陈胜在起义前对士兵的演讲，是中国历史上最早的群众动员文之一：

公等遇雨，皆已失期，失期当斩。藉第令毋斩，而戍死者固十六七。且壮士不死即已，死即举大名耳，王侯将相宁有种乎！

"王侯将相宁有种乎"——这八个字如同惊雷，震碎了秦朝的统治根基。后世对陈胜的评价褒贬不一但不可否认的是，正是陈胜的起义点燃了反秦的燎原之火。贾谊在《过秦论》中评价道："一夫作难而七庙隳"，说的就是陈涉。

陈涉的失败与司马迁的深意

  • 得势后的变质：称王后傲慢待人，"故人皆引去"
  • 用人不当：以朱房为中正，胡武为司过，"诸将以其故不亲附"
  • 战略短视：未能建立稳固根据地，六个月内从称王到被杀
    • 章邯率领骊山囚徒反击
    • 各路义军各自为战不救援

司马迁写陈涉的目的，不只是在写一个人，而是在写一个历史规律：推翻旧秩序的人，往往会被自己建立的新秩序吞噬。陈涉不是唯一的例子——后来的项羽、韩信、彭越……无不如此。

主题切换 JS

function switchTheme(theme) {
  var root = document.documentElement;
  var themes = ['light','dark','sepia','contrast','large'];
  // 清除所有主题 class
  themes.forEach(function(t) {
    root.classList.remove('typo-' + t);
    root.removeAttribute('data-typo-theme');
  });
  // 应用新主题
  if (theme !== 'light') {
    root.setAttribute('data-typo-theme', theme);
    root.classList.add('typo-' + theme);
  }
  // 更新按钮状态
  document.querySelectorAll('.theme-btns button').forEach(function(btn) {
    btn.classList.toggle('active', btn.id === 'btn-' + theme);
  });
}
